Work

  • Posy

    Live | Code
    Built with: Ruby on Rails, React.js, Javascript/jQuery, RSpec

    Based on a website that was wildly popular at my college, users can anonymously post and comment on compliments about others on campus. The page automatically updates when new content is created and features endless scrolling. Posts can be hidden if reported as abusive or removed by an admin.

    Built to learn about React.js as a front-end framework. Interacts with a Rails back-end.

  • ExitSlips

    Live | Code
    Built with: Ruby on Rails, Javascript/jQuery/Coffeescript, RSpec

    Teachers can create short quizzes that are mapped to Common Core standards. Quizzes can be graded automatically or quickly through swiping. Data is automatically compiled. Mastery for each student, class, and standard is displayed and can be exported as a CSV (the lifeblood of many American public school bureaucracies).

    Built in a week with a team of four at Dev Bootcamp in Chicago.

  • Judgy

    Live | Code
    Built with: Laravel (PHP), Javascript/jQuery, Bootstrap, AWS S3, Chart.js, PHPUnit

    Users can upload a profile photo and answer some questions about themselves. Other users can judge the user's answers based only on the uploaded photo. Others' judgments are then displayed for the user.

    Built to learn about PHP and Laravel 4.
